# Elios3 Multi-Sensor Toolkit

Utilities for processing and synchronizing multi-sensor datasets collected with the Flyability Elios 3 UAV.

This repository includes Python scripts for:

- Extracting tracking camera images from ROS1 bag files  
- Inspecting ROS bag topics  
- Extracting RGB frames from video  
- Removing lens distortion from RGB and fisheye images  
- Synchronizing RGB frames with tracking camera timestamps
